# Tile Vision: Flutter Puzzle Game
Developed by **Vijayakumar**

A modern and visually engaging image puzzle game built using Flutter, designed to provide a smooth and interactive user experience across multiple difficulty levels.

---

## 🚀 Features

- **30 Levels** with progressive difficulty
- **Difficulty Tiers**:
  - 🟢 Easy (1–10): 2x2 grid puzzles  
  - 🟡 Medium (11–20): 3x3 grid puzzles  
  - 🔴 Hard (21–30): 4x4 grid puzzles  
- **Modern UI**: Dark theme with glassmorphism design and smooth animations  
- **Guided Hint System**: Step-by-step directional hints (manual solving)  
- **Tile Number Assist Mode**: Toggle to display correct tile positions  
- **In-Game Options**: Skip level, make easy, hints, and controls  
- **Premium Features**: Ad-free experience and extra assists  
- **Audio & Visual Feedback**: Sound effects and confetti animations on level completion

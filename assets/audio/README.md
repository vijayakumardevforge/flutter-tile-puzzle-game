# Audio Files Needed

Due to licensing and generation limitations, audio files are not included.

For best results, add these three files to this directory:
- `click.mp3` - Short click/tap sound (~0.1s)
- `win.mp3` - Victory/success sound (~1-2s) 
- `background.mp3` - Calm ambient music (looping)

You can:
1. Use royalty-free sounds from sites like freesound.org
2. Generate them with tools like jsfxr.com (for retro sounds)
3. Leave empty - the app will work silently

The AudioManager gracefully handles missing files.
